# Arrays vs LinkedList ✨
##
    Space Complexity
* static arrays has fixed size
* dynamic arrays grow by 50-100% (wasting memory)
* linked list less wasting memory than arrays

##
    Space Complexity

|                 | Arrays | Linked List |
| :-------------- | :----: | :---------: |
| <b>Search</b>   |        |             |
| - by index      | O(1) 🔥 |    O(n)     |
| - by value      |  O(n)  |    O(n)     |
| <b>Insert</b>   |        |             |
| - beginning/end |  O(n)  |    O(1)🔥    |
| - middle        |  O(n)  |    O(n)     |
| <b>Delete</b>   |        |             |
| - beginning     |  O(n)  |    O(1)🔥    |
| - middle        |  O(n)  |    O(n)     |
| - end           |  O(n)  |    O(n)     |

<br />

# Singly LinkedList vs Doubly LinkedList ✨
|                 | Singly | Doubly |
| :-------------- | :----: | :----: |
| <b>Search</b>   |        |        |
| - by index      |  O(n)  |  O(n)  |
| - by value      |  O(n)  |  O(n)  |
| <b>Insert</b>   |        |        |
| - beginning/end |  O(1)  |  O(1)  |
| - middle        |  O(n)  |  O(n)  |
| <b>Delete</b>   |        |        |
| - beginning     |  O(1)  |  O(1)  |
| - middle        |  O(n)  |  O(n)  |
| - end           |  O(n)  | O(1)🔥  |
