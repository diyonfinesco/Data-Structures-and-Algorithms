# Arrays vs LinkedList ‚ú®
##
    Space Complexity
* static arrays has fixed size
* dynamic arrays grow by 50-100% (wasting memory)
* linked list less wasting memory than arrays

##
    Space Complexity

|                 | Arrays |    Linked List    |
| :-------------- | :----: | :---------------: |
| <b>Search</b>   |        |                   |
| - by index      | O(1) üî• |       O(n)        |
| - by value      |  O(n)  |       O(n)        |
| <b>Insert</b>   |        |                   |
| - beginning/end |  O(n)  | <mark>O(1)<mark>  |
| - middle        |  O(n)  |       O(n)        |
| <b>Delete</b>   |        |                   |
| - beginning     |  O(n)  | <mark>O(1)</mark> |
| - middle        |  O(n)  |       O(n)        |
| - end           |  O(n)  |       O(n)        |
