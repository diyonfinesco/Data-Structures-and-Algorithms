## Queue (FIFO - First In  First Out)

## Performance 🚀
|               | QueueArray | QueueStack(LL) |
| :------------ | :--------: | :------------: |
| enqueue(item) |    O(1)    |      O(1)      |
| dequeue() 🔥   |    O(1)    |      O(n)      |
| peek() 🔥      |    O(1)    |      O(n)      |
| isEmpty()     |    O(1)    |      O(1)      |
| isFull()      |    O(1)    |       -        |

<br />

## When to use 🤔
* Sharing a resource amongst many customers
